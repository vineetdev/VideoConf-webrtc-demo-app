VIDEO CONFERENCE APP
---------------------------
A Video Conference Application made using WebRTC

This is a video chat application made for beginners to understand the basic functionality of WebRTC.

The front end is pure HTML,CSS,Javascript. The Backend is in Node.js.


Steps to run
------------------------
Clone the repository
Dependencies are all pushed along with the code so you don't need npm install
Run node index.js

USAGE
------------------------
STEP1. Launch app by running on browser http://localhost:4000 
       It will look like below:
       ![VideoRoomAppLandingPage](https://user-images.githubusercontent.com/13249515/209427245-928dc0d1-8541-4e15-9f39-26412162d5f3.jpg)

STEP2. Enter room name.
       Press Join Button.
       On pressing Join button local video preview will start coming as shown below: 
       ![EnteringRoomOneUser](https://user-images.githubusercontent.com/13249515/209427332-6987ab03-8834-4f8b-ba59-331df2dadce2.jpg)

       Viola you have entered the room. 
       
 Other user can enter the same room by following steps 1 to 2 and providing same room name as given by other user in Step2. 
 And both the users should be able to see the other user. 
 ![VideoRoomApp2UsersInRoom](https://user-images.githubusercontent.com/13249515/209427378-f2f583ec-4aad-42b1-a9a2-58004ff02704.jpg)
